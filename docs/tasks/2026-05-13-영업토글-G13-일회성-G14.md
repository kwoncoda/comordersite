## 새 기능 명세 (G13)

### 영업 상태 머신 (2-state)

```
[CLOSED] (default)
   │ 관리자: "🚀 장사 시작" 클릭
   ▼
[OPEN] — 사용자 주문 가능
   │ 관리자: "오늘 정산 마감" 클릭 (ADR-012 가드 통과)
   │ → 정산 스냅샷 생성 + 자동 CLOSED 전이
   ▼
[CLOSED] — 다음날 5/21 다시 "장사 시작" 클릭
```

### 사용자 영향 (CLOSED 상태)

| 경로 | 동작 |
|---|---|
| `/`·`/menu`·`/cart`·`/checkout`·`/orders/:id/*` | 302 redirect → `/closed` |
| `POST /api/orders` | HTTP 423 Locked — `{ error: "BUSINESS_CLOSED" }` |
| `POST /api/orders/:id/transfer-report` | HTTP 423 Locked |
| `GET /api/orders/:id/stream` (SSE) | 423 또는 즉시 close |
| `/admin/*` | 영향 X (관리자는 영업 외에도 작업 가능) |
| `/closed`·`/error/*` | 정상 표시 |
